### wsl 修改安装目录

查看所有分发版本
```cmd
wsl -l --all  -v
```
导出分发版为tar文件到d盘
```cmd
wsl --export Ubuntu-20.04 d:\ubuntu20.04.tar
```
注销当前分发版
```cmd
wsl --unregister Ubuntu-20.04
```

重新导入并安装分发版在d:\ubuntu
```cmd
wsl --import Ubuntu-20.04 d:\ubuntu d:\ubuntu20.04.tar --version 2
```

设置默认登陆用户为安装时用户名
```cmd
ubuntu2004 config --default-user Username
```
删除tar文件(可选)
```cmd
del d:\ubuntu20.04.tar
```
